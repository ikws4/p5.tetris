# How to play?

Just pressed the arrow keys to control the Tetromino :).

Vim user? Then use your favorite vim keybing to control it.

<kbd>h</kbd>: Left

<kbd>j</kbd>: Down

<kbd>l</kbd>: Right

<kbd>k</kbd>: Rotate

<kbd>SHIFT</kbd> + <kbd>j</kbd>: Down with instant
